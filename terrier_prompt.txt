あなたは、画像認識システム「SigmaSense」の設計者です。
これから、よく似た2つの犬種、「ノーフォークテリア」と「ケアーンテリア」を精密に見分けるための、新しい意味次元を設計してください。

要件は以下の通りです。
1.  次元の数は3〜5個程度とします。
2.  各次元は、両者の違いを明確に捉えるための、具体的な特徴量でなければなりません。特に、**耳の形（垂れ耳 vs 立ち耳）** は最も重要な違いです。
3.  各次元は、PythonのOpenCVライブラリを使って具体的なアルゴリズムで計算可能である必要があります。
4.  出力は、JSON形式のリストとしてください。
5.  リストの各要素は、必ず以下のキーを持つJSONオブジェクトでなければなりません。
    - `id`: 次元の一意なID（英語、snake_case）
    - `name_ja`: 日本語の次元名
    - `description`: 次元が何を表すかの簡単な説明
    - `algorithm_idea`: その次元をPythonとOpenCVで計算するための具体的なアルゴリズムのアイデア。画像（NumPy配列、BGR形式）を入力とし、0.0から1.0の範囲の単一の数値を返す前提で、処理のステップを記述してください。

良い例:
{
    "id": "ear_shape_ratio",
    "name_ja": "耳の形状比",
    "description": "耳の輪郭の縦横比。値が大きいほど「立ち耳」であることを示す。",
    "algorithm_idea": "1. Haar Cascade分類器等で犬の顔を検出する。 2. 顔の上部を耳の探索領域(ROI)として設定する。 3. ROI内で輪郭を検出し、面積が最大の2つの輪郭を耳として特定する。 4. 各耳輪郭の境界ボックスの高さと幅を計算し、その比（高さ/幅）を求める。 5. 2つの耳の比の平均値をとり、想定される最大値で正規化する。"
}

上記の要件を厳守し、JSONリストのみを出力してください。
